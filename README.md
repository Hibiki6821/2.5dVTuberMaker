# 2.5D VTuber Maker
https://hibiki6821.github.io/2.5dVTuberMaker/

写真にLive2Dモデルを重ねて2.5D風の画像を生成するWebアプリケーションです。

## 機能

### Live2Dモデル
- Live2DモデルのZIPファイルを読み込み、写真上に配置
- ドラッグで位置調整、スライダーでサイズ・回転を操作
- 顔パラメータ（目・口・頭の向きなど）をスライダーで個別調整
- 円形マスクで切り抜き表示

### AI顔検出
- MediaPipe Face Meshによる顔検出
- 写真内の顔にモデルを自動配置
- バッチモードでは全画像一括検出に対応

### バッチ処理
- 画像ZIPをアップロードして複数画像を一括処理
- 画像ごとにモデル位置・表情・集中線の設定を個別管理
- 選択した画像に表情を一括適用
- 全画像をZIPでまとめて書き出し

### 集中線オーバーレイ
- 集中線画像を重ねて表示（スクリーンブレンド）
- 四隅・四辺のハンドルでドラッグリサイズ
- 中央ドラッグで位置移動
- 幅・高さ・透明度をスライダーで調整
- 画像ごとにON/OFF切り替え可能

### Firebase連携
- モデルをFirebase Storageに保存・読み込み
- 表情プリセットをFirestoreに保存・適用
- マスク設定の自動保存
- 匿名認証による透過的なログイン

## 使い方

1. パスワードを入力してログイン
2. 「写真」から背景画像をアップロード（ドラッグ&ドロップも可）
3. 「モデル(ZIP)」からLive2Dモデルを読み込み、または保存済みモデルを選択
4. 「AI顔検出」で自動配置、またはドラッグ&スライダーで手動調整
5. 表情パラメータを調整
6. 「画像保存」で書き出し

### バッチ処理

1. 複数画像を含むZIPファイルをドロップ
2. 「全画像AI検出」で一括顔検出
3. 画像ごとに表情や集中線を調整
4. 「全画像ZIP」でまとめて書き出し

- Tailwind CSS（CDN）
